## API 使用方法

```python
from AmazingData import login, MarketData, BaseData, constant

# 登录
result = login(USERNAME, PASSWORD, HOST, PORT)

# 获取日历
base = BaseData()
calendar = base.get_calendar()

# 创建 MarketData 实例
market = MarketData(calendar)

# 获取 K 线数据
kline_data = market.query_kline(
    code_list=["600519.SH"],
    begin_date=20250101,
    end_date=20260407,
    period=constant.Period.min60.value  # 60 分钟 K 线
)

# 提取数据
df = kline_data["600519.SH"]
print(f"记录数：{df.shape[0]}")
print(f"字段：{list(df.columns)}")
```

## 注意事项

1. **日期格式**: 必须使用整数格式 (YYYYMMDD)，如 20250101
2. **代码格式**: 必须使用带后缀的格式，如 "600519.SH"
3. **calendar 参数**: 新版本 SDK 必须传入交易日历参数
4. **数据范围**: 单次查询建议不超过 3 个月，长时间范围建议分段查询

---

**测试结论**: AmazingData SDK 的 K 线数据接口工作正常，支持 1 分钟至月线的所有周期，日线数据可获取约 16 年的历史数据。
