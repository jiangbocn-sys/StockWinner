## 银河 SDK 技术细节

### 正确用法
```python
from tgw import IGMDApi, IGMDSpi, Cfg, ApiMode, MarketType, SubscribeItem, Tools_CreateSubscribeItem

# 配置
cfg = Cfg()
cfg.username = "REDACTED_SDK_USERNAME"
cfg.password = "REDACTED_SDK_PASSWORD"
cfg.server_vip = "140.206.44.234"
cfg.server_port = 8600
cfg.force_logout = True

# SPI 回调
class MainSpi(IGMDSpi):
    def OnRspLogin(self, errorMsg): pass
    def OnRtnSnapshotL1(self, session_id, snapshots, count): pass

# 初始化
api = IGMDApi()
ret = api.Init(MainSpi(), cfg, ApiMode.kInternetMode)  # 返回 0 表示成功

# 订阅
items = Tools_CreateSubscribeItem(1)
items.market = MarketType.kSSE
items.security_code = "600519"
ret = api.Subscribe(items, 1)  # 返回 0 表示成功
```

### 错误码
| 错误码 | 含义 |
|--------|------|
| 0 | 成功 |
| -97 | 参数非法 |
| -98 | 空指针 |
| -99 | 未初始化 |

### 市场类型
